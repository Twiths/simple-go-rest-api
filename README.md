# simple-go-rest-api
A rest api in GO and MongoDB

### Tech stack
- MongoDB as Database
- Fibre (monogdb driver)
- GO

### How to run
```bash
Run go build
```

### Endpoints
```
GET /person/:id? - Get a person by ID
POST /person - Post a person
PUT	/person/:id - Update a person by a given ID
DELETE /person/:id - Delete a person by a given ID
```
